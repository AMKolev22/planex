## 📥 Installation
<p>First you have to clone our project <b>(nodejs required)</b>: </p>
<pre>git clone https://github.com/codingburgas/2324-space-sprint-project-planex.git</pre>
<br>
<p>Open it in your favourite code editor or IDE, build a new terminal and <b>type the following to start the server: </b> </p>
<pre>
cd app
cd renderer
npm i
npm run dev
</pre>
<br>
<p>And finally, start the app: </p>
<pre>
cd app
npm i
npm run start
</pre>
